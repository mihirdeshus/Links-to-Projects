Mihir Deshpande — Selected Work

1. UpLift21 — AI Prenatal Screening System

GitHub: github.com/mihirdeshus/UpLift21

What it is: An AI-assisted screening framework for early detection of Down syndrome, designed to run on a Raspberry Pi (edge device) with no cloud dependency.

Problem: Early screening is expensive and inaccessible in low-resource settings.

My contribution: Built the full ML pipeline using a gradient boosting ensemble on 7 clinical biomarkers, optimized for <200ms inference, and designed for real-world usability in rural clinics.

2. Accessibility Toolkit — Browser Extensions

GitHub:

  • Color Blindness Filter
  • Custom Font Changer

What it is: A suite of browser extensions for color correction and readability enhancement.

Problem: Many websites are not accessible to users with visual impairments.

My contribution: Built both extensions end-to-end using JavaScript and CSS filters, focusing on real-time transformations across arbitrary websites.

3. Kaelo — Blockchain Livestock Management

GitHub: github.com/ChumaniC/Kaelo

What it is: A blockchain-based livestock tracking system built on Hedera.

Problem: Lack of reliable livestock records reduces trust and compliance for farmers.

My contribution: Worked on system design and front-end development, and helped integrate smart contract logic into a usable interface.
