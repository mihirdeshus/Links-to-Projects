1. UpLift21 — AI Prenatal Screening System

GitHub: github.com/mihirdeshus/UpLift21

UpLift21 is an AI-assisted screening framework for early detection of Trisomy 21 (Down syndrome), designed explicitly for deployment in resource-constrained healthcare environments. Unlike conventional approaches that assume access to cloud infrastructure, high-end imaging systems, and trained specialists, this system is built to operate entirely on a sub-$50 Raspberry Pi connected to legacy ultrasound machines — with no internet and no on-site expertise required.

The problem it addresses is not just medical, but infrastructural: in many rural settings, early screening is effectively inaccessible because the existing pipeline depends on scarce human expertise and expensive, centralized systems. UpLift21 rethinks this pipeline from first principles.

I designed and implemented the system end-to-end, developing a gradient boosting ensemble trained on seven clinically relevant biomarkers (including NT, nasal bone, FHR, serum markers, maternal age, and CRL). The model is optimized for edge deployment, achieving <200ms inference while maintaining a ROC-AUC of 0.979 at an approximate cost of $15 per screen. The central design constraint was usability — ensuring the system could be meaningfully operated by non-specialists in primary health centres without disrupting existing workflows.

The project originated in Prof. Manu Prakash’s Frugal Science course at Stanford and is currently in discussion for a $10,000 research grant. It was awarded 1st Place at TechSparks 2026 at Ashoka University.

2. Kaelo — Blockchain Livestock Management

GitHub: github.com/ChumaniC/Kaelo

Kaelo is a blockchain-based livestock management system designed to address a fundamental gap in agricultural infrastructure: the absence of reliable, verifiable records for small-scale livestock farmers. In many emerging markets, particularly within Local Support Model (LSM) groups in southern Africa, farmers lack tamper-proof documentation of animal health, vaccination history, and compliance.

Developed during the University of Zurich’s Deep Dive into Blockchain programme, Kaelo leverages Hedera-based smart contracts to create immutable, auditable livestock records across the lifecycle of an animal — including identification, veterinary interactions, and compliance.

My contribution spanned both system design and implementation. I structured the data model and built the complete front-end (HTML/CSS/JavaScript), creating interfaces for farmers, veterinarians, and administrators. The focus was on usability — ensuring the system could function intuitively in low-resource, real-world contexts.

3. Accessibility Toolkit — Browser Extensions

GitHub:

  • Custom Font Changer
  • Color Blindness Filter

This project is a two-part accessibility toolkit consisting of lightweight browser extensions designed to improve usability of the web for individuals with visual impairments and reading difficulties.

The Color Blindness Filter applies real-time color transformations using JavaScript and CSS filters, while the Custom Font Type & Size Changer enables dynamic typography adjustments on any live webpage.

I built both tools independently with a focus on performance, universality, and zero-friction usability. They work seamlessly across arbitrary websites with minimal overhead and no setup required.

The project advanced to the finals of the Empower Hackathon at IIM Ahmedabad (March 2025).
